# Frontend Application

Frontend for AI Mock Interview application - vanilla JavaScript with real-time speech recognition and video.

## Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the frontend server:**
   ```bash
   npm start
   ```
   Frontend will run on `http://localhost:3000`

   The frontend expects the backend server to be running on `http://localhost:5000`

## Features

- Resume PDF upload and parsing (PDF.js)
- Real-time speech recognition (Web Speech API)
- Live video capture with camera toggle
- AI-powered question generation
- Interactive interview experience
- Performance analysis and reporting

## Technologies

- Vanilla JavaScript (no frameworks)
- PDF.js for resume parsing
- Web Speech API for speech recognition
- HTML5 audio for text-to-speech playback
- Modern CSS (Grid, Flexbox, animations)
